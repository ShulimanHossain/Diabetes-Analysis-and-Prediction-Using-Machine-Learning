#  Diabetes Analysis and Prediction Using Machine Learning

##  Introduction

This is a practice project focused on applying Data Science and Machine Learning techniques to the problem of diabetes prediction.  
The project uses a publicly available dataset obtained from Kaggle that simulates real-world medical and lifestyle data.

Through exploratory data analysis (EDA) and supervised classification models, the goal is to strengthen practical skills in:
- Data preprocessing  
- Feature analysis  
- Healthcare-focused predictive modeling
